#### 为什么前端需要构建？
- 开发复杂化
- 框架去中心化
- 语言编译化
- 开发模块化

#### 为什么webpack？
- 三大框架的脚手架基于webpack，如：vue-cli、react-starter、angular-cli
- 代码分割：code-splitting
- 天生的模块化
